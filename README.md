# reversi-game-in-elm
Reversi Game implemented in Elm (https://en.wikipedia.org/wiki/Reversi)

DEMO: http://inderps.github.io/reversi-game-in-elm/

### Development
* Run `npm install`
* Run `elm-package install --yes`
* Run `npm start`
* Go to `localhost:8080` to see the app

### Production Deployment
* Run `npm run build`
